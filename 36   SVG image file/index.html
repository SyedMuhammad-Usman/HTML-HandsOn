SVG Example

SVG stands for Scalable Vector Graphics. It is an XML-based markup language for describing two-dimensional vector graphics. SVG is used to define graphics for the web that are scalable and can be manipulated via CSS and JavaScript.

Explanation of the Example: svg element: Defines the SVG container. width and height: Define the dimensions of the SVG container. viewBox: Defines the coordinate system of the SVG. xmlns: Specifies the XML namespace for SVG. circle element: Defines a circle. cx and cy: Define the coordinates of the circle's center. r: Defines the radius of the circle. stroke: Specifies the color of the circle's outline. stroke-width: Specifies the width of the circle's outline. fill: Specifies the fill color of the circle.
