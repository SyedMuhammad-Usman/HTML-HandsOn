Block and Inline Elements in HTML

Block Elements:

Block elements typically start on a new line and take up the full width available. - Examples include headers, paragraphs, divs, and more.

Inline Elements:

- Inline elements do not start on a new line and only take up as much width as necessary. - Examples include links, spans, and text formatting tags.

List of Block Elements:

- `<div>` - `<p>` - `<h1>` to `<h6>` - `<ul>` - `<ol>` - `<li>` - `<blockquote>` - `<pre>` - `<address>` - `<form>` - `<table>` - `<header>` - `<footer>` - `<section>` - `<article>` - `<nav>` - `<aside>` - `<figure>` - `<figcaption>` - `<main>`

List of Inline Elements:

- `<a>` - `<span>` - `<b>` - `<i>` - `<strong>` - `<em>` - `<img>` - `<br>` - `<code>` - `<sub>` - `<sup>` - `<small>` - `<time>` - `<mark>` - `<q>` - `<cite>` - `<abbr>` - `<acronym>` - `<data>` - `<kbd>` - `<samp>` - `<xmp>` These lists include many of the common block and inline elements used in HTML, though there are others that can also be categorized under each type.
